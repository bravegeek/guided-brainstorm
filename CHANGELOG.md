# Changelog

## v0.1.0 — 2026-04-21

First public release.

- Four-phase Guided Brainstorm framework: Situation / Advisor / Conversation / Plan / Save
- `SKILL.md` for Claude Code (agentic, writes session files to `./brainstorms/`)
- `prompt.md` for paste-anywhere use in ChatGPT, Gemini, and other LLM chat interfaces (outputs inline)
- Eight brainstorming categories spanning tech, business, creative, personal, academic, and career domains
